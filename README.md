# ☕ Brew Haven Café – SEO Content Cluster Project

## 📌 Overview
This project demonstrates an AI-powered SEO strategy designed for Brew Haven Café to improve Google rankings, local visibility, and organic traffic.

---

## 🎯 Objectives
- Improve local SEO ranking in Chennai
- Target high-intent café-related keywords
- Build structured content clusters
- Increase organic traffic through blogs

---

## 🧠 SEO Strategy Used
- Keyword Research (Primary, Secondary, Long-tail)
- Pillar + Cluster Content Model
- Internal Linking Strategy
- AI-assisted blog generation

---

## 🏗️ Project Structure
- Keyword Research File
- Content Cluster Map
- Pillar Blog (Main SEO Page)
- Supporting Blog Articles
- SEO Strategy Document
- Monetization Plan

This structure can be used by cafés, restaurants, and local businesses to:
- Rank on Google Search
- Attract nearby customers
- Build online presence
